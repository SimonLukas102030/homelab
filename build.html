Hardware Build — three tiers

Three complete builds, all in a 19" rack: server, switch (Phase 1: TP-Link TL-SG2210P, 8-port PoE, rack-native 1U — Cisco SF 300-48p goes to the barn for PoE cameras), rack frame, switched PDU, edge box. Medium and Pro include a GPU for local AI. Realistic used prices from eBay / Kleinanzeigen / ServerShop24, as of April 2026.

TIER 1Entry — under €1,000, no GPU

Dell PowerEdge R720 (2014, DDR3 platform). Sufficient for 2× MC servers, Plex, Nextcloud, Portainer, and a stack of small services. Local AI can be added later as an upgrade.

Server: Dell PowerEdge R720 (2U) ~€220 used

With 2× Xeon E5-2680v2 (10 C / 20 T per CPU = 20 C / 40 T total) and 64 GB DDR3-1866 ECC, base configs start at ~€180–250 on eBay/Kleinanzeigen. Get the model with 8× 2.5" bays (LFF variant has 8× 3.5") and H710 RAID controller in IT mode (for Proxmox ZFS).

What to watch for: RAM slots are in 3 banks per CPU — for DDR3-1866 speed use 8 DIMMs (4 per CPU), not 12. The PERC H710 must be flashed to "HBA mode" / IT mode for ZFS — otherwise ZFS tools don't see the disks directly.

RAM: 64 GB DDR3 ECC (default) — upgrade to 128 GB optional ~€100 / +€100

64 GB default = 4× 16 GB DDR3-1866 RDIMM ECC ~€100 used (~€20–30 per module). Enough for 2× MC + Plex + Nextcloud + stack with headroom. Upgrade to 128 GB (4× additional 16 GB DIMMs) later ~+€100. Search: "DDR3 16GB 1866 ECC RDIMM" or "PC3-14900R". Brands: Samsung, Hynix, Micron — all equally good, avoid no-name.

Pro tip: An R720 with RAM already installed (e.g. R720 + 128 GB DDR3 as a bundle for €350–450) is often cheaper than a bare R720 (€220) + 128 GB RAM separately (€200) = €420. Filter Kleinanzeigen specifically for "R720 128GB".

Storage: 1 TB NVMe + 2× 4 TB SAS HDD ~€170

NVMe (Lexar NM790 1 TB, ~€80) via PCIe adapter (~€10) — the R720 has no M.2 slot, but plenty of PCIe slots. For Proxmox host + VM roots + Plex cache. 2× 4 TB SAS HDD used (~€80 combined) → ZFS mirror on the drive bays for Plex/Nextcloud/backups. SAS over SATA: in the server world, cheaper used with the same performance.

Rack: 9U open-frame StarTech / Triton ~€140

9U is enough: 2U server + 1U switch + 1U patch panel + 1U PDU + 1U cable manager + 3U reserve. Open-frame over enclosed → better airflow, ~€100 cheaper. Depth 600 mm, the R720 is 67 cm deep — tight but fits.

Alternative: Triton 6U wall-mount (~€90) if you want to save floor space — but then an R720 (2U) + switch (1U) + reserve barely fits. Not expandable.

Switched PDU: APC AP7900B used ~€100 used

8 outlets individually switchable via web UI / SNMP / SSH. Exactly what you wanted ("turn individual devices on/off"). Used on eBay ~€80–150, new ~€280. Check battery & firmware version on used units. PDU self-consumption: ~3 W.

Cheap alternative (no rack mount): Brennenstuhl Premium-Line 8-way with individual switches (~€40) — manual switching, no remote. Smart alternative: 8× TP-Link Tapo P100 smart plugs (~€60) — app-switchable, not rack format but very flexible.

Edge box: existing Dell Optiplex DDR3 €0

Your existing Optiplex (16 GB DDR3) is too weak as a main server — but perfect as an edge server: always-on, ~15 W idle, hosts the Cloudflare tunnel + WoL trigger to wake the main server. Puts a machine to use that you would otherwise have thrown away.

Alternative new: Raspberry Pi 4 8 GB (~€80) — only 3 W idle. But free Optiplex beats paying for a Pi.

Fan mod: 4× Noctua NF-A6x25 PWM + cable adapters ~€80

The R720 has 6 stock 60 mm fans that idle at 8,000+ RPM. Bedroom torture. Mod: replace 60 mm slots with Noctua NF-A6x25 PWM (60 mm! ~€18/ea) — quiet (3000 RPM max) instead of a noise disaster. Plus 4-pin to Dell adapter cables (~€15 from modders on eBay) — Dell uses a proprietary 5-pin connector.

Required add-on: IPMI fan override (free via ipmitool) — otherwise iDRAC spins the mod fans up anyway when it detects "non-Dell" fans. Details: Operations.

Patch panel + Cat6 cables + cable manager ~€60

24-port Cat6 keystone patch panel (1U, ~€30) + 8× Cat6 patch cables 0.5 m (~€15) + 1U horizontal cable manager (~€15). Keeps the rack tidy and clearly maps which port goes where.

TIER 2 ★Medium — Best price/performance, with local AI

Dell PowerEdge R730 (DDR4) + Tesla P40 24 GB for local AI. More power-efficient than entry tier, GPU for LLMs & Stable Diffusion, more RAM headroom for future projects.

Server: Dell PowerEdge R730 (2U) ~€380 used

2× Xeon E5-2680v3 (12 C / 24 T per CPU = 24 C / 48 T total) with 64 GB DDR4-2133 ECC. On eBay/Kleinanzeigen ~€350–500 used in reasonable config. Search for R730 with "8x 2.5" or "8x 3.5" bays + iDRAC 8 Express. Generational jump over R720: ~30% lower idle wattage, DDR4 over DDR3, faster PCIe 3.0, GPU support out of the box.

Important for GPU: you need the GPU-enabled version (with GPU riser bracket + 1100 W PSU). Ask the seller or request an interior photo — the riser is a large metal frame on the right side.

RAM: 128 GB DDR4 ECC RDIMM ~€400

4× 32 GB DDR4-2400 RDIMM ECC ~€80–120 per module. 128 GB kit = ~€340–460 used. Alternatively 8× 16 GB ~€280–360 (smaller modules often cheaper per GB, but all 8 slots occupied = harder to expand later). Search: "DDR4 32GB 2400 ECC RDIMM", "PC4-19200R" or "PC4-21300R" (2666). Brands: Samsung M393A4K40, Hynix HMA84GR7, Micron MTA36ASF — all the same. On the R730 specifically use compatible RAM — check Dell RDIMM compatibility lists, some no-name modules simply won't boot.

Pro tip: R730 with 64 GB DDR4 already installed as a bundle (~€450–550) plus a 64 GB expansion (~€200) = €650–750, often cheaper than bare R730 + 128 GB separately.

Local AI GPU: Tesla P40 24 GB ~€220 used

Server GPU with 24 GB GDDR5 — fits in the R730 via the GPU riser bracket. Pascal architecture (Compute 6.1) — works fine with Ollama / llama.cpp / vLLM. Performance: Llama 3 70B (Q4): ~6 t/s; Llama 3 8B: ~23 t/s. Sweet spot is 14B models in Q4 (Phi-4, Qwen 2.5 14B) — fit fully in 24 GB with context headroom.

Cooling mod required: P40 is passive (server form factor expects server airflow). You need a 3D-printed shroud + 80 mm fan (standard mod, templates freely available on Thingiverse, ~€20 for fan + optional print). Power connector: Dell riser has 5 GND pins, P40 has 4 GND — wrong connector = short-circuit protection trips, server won't boot. Use an adapter cable with the correct pinout (~€15 on eBay "Dell P40 power adapter").

Storage: 1 TB NVMe + 4 TB HDD ~€130

Lexar NM790 1 TB (~€80) via PCIe adapter — faster than the entry-tier 512 GB variant (more Plex cache, more VM storage). 1× 4 TB HDD WD Red Plus (~€110) for /mnt/bulk — no mirror in Medium tier.

Rack: 12U open-frame ~€200

12U over 9U: more headroom for future expansion (second server, NAS, switch upgrade). Depth 800 mm — gives room behind the R730 (67 cm) for cable management without strain. StarTech 4POSTRACK12U or equivalent.

Switched PDU APC AP7900B used ~€140 used

Same model as entry tier, but a newer unit (fresher firmware, better battery). Each of the 8 outlets switchable via web/SSH — you can isolate the MC server VM from the Plex server circuit, or disable the GPU for power-saving when not in use.

Edge box: reuse Optiplex OR Pi 4 8 GB €0 or ~€80

Same as entry tier. Pi 4 8 GB is ~12 W less than the Optiplex — over a year ~€30 less in electricity. If you'd otherwise throw the Optiplex away, keep using it.

Fan mod: 6× Noctua NF-A6x25 PWM + adapters ~€120

R730 has 6 fan slots. Plus the GPU cooling solution (see above). Total fan budget ~€120 (server) + ~€20 (GPU shroud) = €140.

Patch panel + Cat6 + cable mgmt ~€60

Same as entry tier. Scales without issues.

TIER 3Pro — Full build for demanding workloads

Dell PowerEdge R740 (newer, quieter, GPU-optimized) + RTX 3090 (24 GB, faster than P40, gaming-capable). 256 GB RAM, ZFS mirror on NVMe, UPS, enclosed 15U rack. For when you really want to go all in.

Server: Dell PowerEdge R740 (2U) ~€800 used

14th Gen — 2× Xeon Gold 6140 (18 C / 36 T per CPU = 36 C / 72 T total), 128 GB DDR4-2666 ECC. Used on eBay €700–950 in reasonable config. Big advantage for AI: R740 has dedicated GPU risers for 4× FH/FL GPUs or 2× double-wide. PCIe Gen3 ×16 without bifurcation tricks.

8× 32 GB DDR4-2666 RDIMM ECC ~€90–130 per module. 256 GB kit ~€720–900 used. Largest single line item in the Pro build. Alternative: start with 128 GB (~€400) and expand later — saves €380 now.

256 GB only makes sense if you really host 4+ MC servers simultaneously, or multiple AI workloads in parallel (LLM inference + SDXL batch). For 95% of homelab use cases 128 GB is enough. Verdict: starting with 128 GB (Pro total drops to ~€3,305) is the pragmatic route.

Local AI GPU: RTX 3090 24 GB used ~€650 used

RTX 3090 has the same 24 GB VRAM as P40, but: ~3× faster in inference (GDDR6X over GDDR5, Ampere over Pascal), FP16-capable without tricks, lower power per FLOP, usable for gaming. 350 W TDP over 250 W — 1100 W PSU is mandatory. Used on eBay €600–750.

3090 over P40 advantages: SDXL generation in 5 s instead of 30 s, Llama 70B at Q5 instead of Q4 (better quality), audio/Whisper noticeably faster, Flux image gen works. Downside: ~€430 more expensive, louder (own fans, not passive). Alternative: 2× P40 = 48 GB total for ~€440 — handles 70B at Q8 (max quality).

Storage: 2× 1 TB NVMe (mirror) + 2× 4 TB HDD (mirror) ~€350

2× Samsung 990 Pro 1 TB as ZFS mirror = redundancy if one NVMe dies. 2× 4 TB WD Red Plus as ZFS mirror for /mnt/bulk = data survives a single HDD failure. Usable storage: 1 TB NVMe + 4 TB HDD.

Rack: 15U enclosed with glass door ~€350

Lanview / Triton 15U closed with front glass + side panels. Looks like a professional datacenter component. Important: active ventilation required (top-mounted roof fan tray, ~€60 extra) — an enclosed rack without airflow will heat-soak.

Smart PDU with per-outlet metering: APC AP8941 ~€220 used

Switchable AND metered per outlet (watts + current + voltage). You can see exactly how much the server vs. GPU vs. switch draws — perfect for power tuning. Plus: programmable power sequences (e.g. switch first, then server, then GPU).

UPS: APC Smart-UPS 1500VA used ~€200 used

Online sine-wave UPS with 980 W output. Keeps server + switch alive ~30 min on power loss — time for a clean shutdown via NUT. Battery check mandatory on used purchase (battery > 4 years = replace, ~€80 replacement). USB port for NUT integration with Proxmox.

Edge box: Raspberry Pi 5 8 GB ~€100

Pi 5 over Optiplex: 3 W idle, modern, ARM-efficient. Hosts cloudflared + WoL trigger. Pi 5 has fast I/O, NVMe hat option (for logs/cache).

Premium fan mod: Noctua NF-A6x25 G2 + GPU cooling ~€140

Premium Noctua fans (G2 = 2024 generation). RTX 3090 has its own fans (triple-slot), no additional cooling needed — but case airflow must be adequate. Plus Noctua NT-H2 thermal paste refresh on the CPU heatsink.

8-port Gigabit backbone switch: TP-Link TL-SG108E ~€35

Since the Cisco SF 300-48p only delivers 100 Mbit on the 48 access ports: a small Gigabit switch in front for server + workstation + edge box. Managed (VLAN-capable) for clean segmentation.

Patch panel 24-port + Cat6a cables + cable mgmt ~€80

Cat6a over Cat6 (for 10G upgrade later), 24-port patch panel, more patch cables for the additional devices.
