NOCTUAFan Mod — Mandatory for the Bedroom

🔊
Stock fans are intolerably loud: R720/R730 run at 8,000+ RPM out of the box — over 50 dB(A). Without the mod the server is not sleep-compatible in a bedroom. With Noctua mod: ~30–35 dB(A) idle.

What gets installed

6× Noctua NF-A6x25 PWM 6 × 17 € = 101 €

R720/R730 have 6 hot-swap fan slots each 60×60×38 mm. Noctua doesn't make 38 mm deep fans — NF-A6x25 (60×60×25 mm) is 13 mm shorter but physically fits in the bay. The reduced airflow is compensated by IPMI temperature control.
Amazon: amazon.de — NF-A6x25 PWM  |  Geizhals: Price comparison

Option A — Adapter cable (easier)

Ready-made "Dell R720 Fan Adapter → 4-Pin PWM" cables from eBay, ~5 €/piece. Plug in, done. No soldering.

Search: "Dell R720 fan adapter Noctua" on eBay.de

Option B — Re-pin yourself (cheaper)

Cut the Dell connector off the old fan leaving ~5 cm of cable, splice onto the Noctua cable using heat-shrink solder connectors. No soldering iron needed, ~2 € material per fan.

Guide: Medium — Silent Dell Server (pinout photos)

IPMI Fan Control — required after the mod

⚠
iDRAC spins up to 100% when it detects non-Dell fans. IPMI manual mode disables automatic thermal protection — keep an eye on temperatures yourself!
# Enable manual mode ipmitool -I lanplus -H <iDRAC-IP> -U root -P <password> raw 0x30 0x30 0x01 0x00 # Set fan speed to 20% (0x14 = 20 decimal) ipmitool -I lanplus -H <iDRAC-IP> -U root -P <password> raw 0x30 0x30 0x02 0xff 0x14 # Re-enable automatic control (e.g. under heavy load) ipmitool -I lanplus -H <iDRAC-IP> -U root -P <password> raw 0x30 0x30 0x01 0x01
🔒
Freeze iDRAC firmware! IPMI fan control only works up to firmware 3.30.30.30. Newer versions block the raw command. Check the current firmware version before the first update — if it's compatible, do not update.
